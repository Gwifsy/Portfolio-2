Mission Implémentez le front-end d'une application bancaire avec React

Étapes du projet :

Etape 1: Authentification des utilisateurs.

Etape 2: Mettez en place le state global avec Redux et la connection de l’utilisateur.

Etape 3: Modifiez le nom d’utilisateur via le formulaire.

Etape 4: Analyse des maquettes des transactions.

Etape 5: Écrivez les différents endpoints dans le fichier Swagger.

Acquis :

1 ) Afficher les données du back end sur l'interface via des appels API. ✅

2 ) Configurer des routes API pour la communication client / serveur. ✅

3 ) Implémenter des données avec Redux pour assurer le fonctionnement du Frontend. ✅
